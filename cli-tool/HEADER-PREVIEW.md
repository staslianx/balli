## Color Scheme

- **"balli" ASCII art**: Purple/Magenta (colors.metric)
- **Box border**: Cyan (borderColor: 'cyan')
- **Title**: White bold (colors.highlight)
- **Subtitle**: Gray (colors.meta)

## Font Details

- **Font**: DOS Rebel (figlet)
- **Style**: Lowercase "balli"
- **Horizontal Layout**: Default
- **Vertical Layout**: Default

## Commands

Both commands work identically:

```bash
# Original command
research-xray

# Short alias
balli-x
```

After running `npm link`, you can use `balli-x` globally from any directory!
